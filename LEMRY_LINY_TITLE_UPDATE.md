## ?? How It Works

### Title Priority:

1. **Backend Config** - If `nonRegularsTitle` is set in `appsettings.json`
2. **Frontend Fallback** - If not set, uses "Lemry líný"

```typescript
{leaderboard.nonRegularsTitle || 'Lemry líný'}
```

### Current Setup:

**Backend (`appsettings.Development.json`):**
```json
"NonRegularsTitle": "Nepravidelni"
```

**Frontend (fallback):**
```typescript
'Lemry líný'  // Now shows if backend title is empty
```

**Result:** Currently shows **"Nepravidelni"** from backend config

---

## ?? To Show "Lemry líný"

### Option 1: Update Backend Config (Recommended)

**File:** `Elo-fotbalek/appsettings.Development.json`

```json
"NonRegularsTitle": "Lemry líný"
```

Then restart backend (Press F5)

### Option 2: Remove Backend Config

**File:** `Elo-fotbalek/appsettings.Development.json`

```json
"NonRegularsTitle": ""  // Empty string uses frontend fallback
```

Then restart backend (Press F5)

## ?? Meaning

### "Lemry líný"

- **Lemry** - Probably a playful/local term (lemurs?)
- **líný** - Lazy (in Czech)
- **Full meaning:** "Lazy lemurs" or similar playful nickname

### Context:
Players who attend less than 30% of matches get this fun title instead of the more formal "Nepravidelní hrá?i" (Non-regular players).
