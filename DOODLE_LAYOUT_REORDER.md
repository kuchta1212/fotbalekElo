## Design Improvements

### Compact Team Generator

**Key Changes:**
1. **Title includes date:** "Vygenerovat týmy pro 23.01"
2. **Flexbox layout:** `flex-col md:flex-row md:items-center md:justify-between`
3. **Inline controls:** Season buttons + Generate button on same line (desktop)
4. **Smaller padding:** `p-4` instead of `p-6`
5. **Removed description:** No extra explanatory text (obvious from context)

**Responsive Behavior:**
- **Desktop (?768px):** 
  - Title on left
  - Season + Button on right
  - All in one row
- **Mobile (<768px):** 
  - Stacks vertically
  - Still compact

### Visual Hierarchy

**Priority order (top to bottom):**
1. ?? **High:** Stats (immediate info)
2. ?? **High:** Generate Teams (primary action)
3. ?? **Medium:** Date selector (context)
4. ? **Low:** Player list (detailed interaction)
