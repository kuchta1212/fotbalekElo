### 1. ? Changed Click Order

- **1st click (from NoAnswer):** ? **Accept** (P?ijde)
- **2nd click:** ?? **Maybe** (Možná)  
- **3rd click:** ? **Refused** (Nep?ijde)
- **4th click:** ? **Accept** (cycles back)

**Before:**
```
NoAnswer ? Refused ? Accept ? Maybe ? Refused
```

**After:**
```
NoAnswer ? Accept ? Maybe ? Refused ? Accept
```

**Code change:**
```typescript
if (currentStatus === 'NoAnswer') {
  nextStatus = 'Accept'; // First click = Accept
} else {
  const statusCycle: DoodleStatus[] = ['Accept', 'Maybe', 'Refused'];
  const currentIndex = statusCycle.indexOf(currentStatus);
  nextStatus = statusCycle[(currentIndex + 1) % statusCycle.length];
}
```
