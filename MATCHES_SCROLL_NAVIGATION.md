# ? Matches Section - Scroll Instead of Navigate

## ?? What Was Changed

Instead of creating a separate matches page, clicking "Z·pasy" in the navigation now **scrolls to the matches section** on the leaderboard (home page).

---

## ?? Changes Made

### 1. Navigation Component

**Added scroll functionality:**

```typescript
const scrollToMatches = () => {
  setIsMenuOpen(false);
  if (location.pathname !== '/') {
    // If not on home page, navigate home first
    window.location.href = '/#matches';
  } else {
    // Already on home page, just scroll
    const matchesSection = document.getElementById('matches');
    if (matchesSection) {
      matchesSection.scrollIntoView({ behavior: 'smooth', block: 'start' });
    }
  }
};
```

**Replaced Link with Button:**

**Before (Desktop):**
```tsx
<Link to="/matches">Z·pasy</Link>
```

**After (Desktop):**
```tsx
<button onClick={scrollToMatches}>
  Z·pasy
</button>
```

**Before (Mobile):**
```tsx
<Link to="/matches" onClick={() => setIsMenuOpen(false)}>
  Z·pasy
</Link>
```

**After (Mobile):**
```tsx
<button onClick={scrollToMatches}>
  Z·pasy
</button>
```

### 2. HomePage - Matches Section

**Added ID and scroll offset:**

**Before:**
```tsx
<Card className="...">
```

**After:**
```tsx
<Card id="matches" className="... scroll-mt-20">
```

- `id="matches"` - Allows scrolling to this element
- `scroll-mt-20` - Adds top margin when scrolling (accounts for sticky nav)

### 3. Card Component

**Added support for id prop:**

**Before:**
```typescript
interface CardProps {
  children: ReactNode;
  className?: string;
}

export function Card({ children, className }: CardProps) {
  return <div className={...}>{children}</div>;
}
```

**After:**
```typescript
interface CardProps {
  children: ReactNode;
  className?: string;
  id?: string;  // ? Added
}

export function Card({ children, className, id }: CardProps) {
  return <div id={id} className={...}>{children}</div>;
}
```
